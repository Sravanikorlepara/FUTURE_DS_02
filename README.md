# Customer Retention & Churn Analysis

## Project Overview

This project analyzes customer retention and churn behavior using the Telco Customer Churn dataset. The objective is to identify churn patterns, customer lifetime trends, and key retention drivers to support better business decisions.

## Tools Used

- Microsoft Power BI
- Microsoft Excel

## Dataset

Telco Customer Churn Dataset

## Dashboard Features

- Total Customers
- Churned Customers
- Churn Rate
- Average Customer Tenure
- Churn by Contract Type
- Customer Distribution by Internet Service
- Churn by Payment Method
- Average Monthly Charges by Churn
- Average Tenure by Churn
- Interactive Slicers

## Key Insights

- Month-to-month customers have higher churn.
- Customers with shorter tenure are more likely to leave.
- Higher monthly charges are associated with increased churn.
- Contract type significantly impacts customer retention.

## Recommendations

- Promote long-term contracts.
- Offer loyalty rewards.
- Improve customer engagement.
- Monitor churn using analytics dashboards.
